## 一、棋盘格子 (GridItem)

### 尺寸

| 参数 | 值 | 设定位置 |
|---|---|---|
| 格子大小 (cellSize) | **70×70 px** | `game.cpp:24` 构造函数 `m_geometry(Board::ROWS, Board::COLS, 70)` |
| 格子行列数 | **8×8 = 64 个** | `board.h:12-13` `ROWS=8, COLS=8` |
| 边框线宽 | **2 px** | `griditem.cpp:35` `QPen(border, 2)` |
| 边框内缩 | **1 px**（防止边框被邻居裁剪） | `griditem.cpp:37` `QRectF(1, 1, cellSize-2, cellSize-2)` |
| 圆角 | 无（直角矩形） | — |

### 绘制位置

每个格子通过 `setPos()` 定位到场景坐标：

```cpp
// game.cpp buildScene()
gridItem->setPos(
    m_geometry.gridToWorld(row, col).x() - m_geometry.cellSize() / 2.0,
    m_geometry.gridToWorld(row, col).y() - m_geometry.cellSize() / 2.0
);
```

`gridToWorld(row, col)` 返回格子中心点，减去半个 cellSize 得到左上角。

### 上半场/下半场颜色

```cpp
// game.cpp buildScene()
gridItem->setBaseColor(
    row < Board::ROWS / 2
        ? QColor(80, 60, 60)   // 上半场 (敌方) = 暗红
        : QColor(60, 60, 80)   // 下半场 (我方) = 暗蓝
);
```

### 高亮状态 (griditem.cpp paint)

| 状态 | 填充色 | 边框色 |
|---|---|---|
| 默认 | m_baseColor | (40, 40, 40) 深灰 |
| 悬停 (hover) | m_baseColor.lighter(120%) | (220, 220, 220) 亮灰 |
| 合法落点 (drop) | (110, 170, 110) 绿色 | (100, 255, 100) 亮绿 |

## 二、棋盘整体布局 (BoardGeometry)

### 坐标系

| 参数 | 值 | 设定位置 |
|---|---|---|
| cellSize | **70 px** | `boardgeometry.h:11` 构造函数参数 |
| 棋盘总宽 | 8 × 70 = **560 px** | — |
| 棋盘总高 | 8 × 70 = **560 px** | — |
| 棋盘底部 Y | `m_rows * m_cellSize` = **560 px** | `boardgeometry.cpp:28,45` |

### gridToWorld（逻辑坐标 → 像素）

```cpp
// boardgeometry.cpp:10-11
x = col * cellSize + cellSize / 2.0   // 例: col=3 → 3×70+35 = 245 (格子中心X)
y = row * cellSize + cellSize / 2.0   // 例: row=5 → 5×70+35 = 385 (格子中心Y)
```

### worldToGrid（像素 → 逻辑坐标）

```cpp
// boardgeometry.cpp:17-18
col = world.x() / cellSize    // 直接除法
row = world.y() / cellSize
// 结果 clamp 到 [0, 7]
```

### 槽位矩形绘制 (game.cpp buildScene)

```cpp
// game.cpp:107-122
QGraphicsRectItem* slot = new QGraphicsRectItem{
    center.x() - 35,   // 左上角 X (70/2)
    center.y() - 40,   // 左上角 Y (80/2)
    70,                // 宽
    80                 // 高

slot->setPen(QPen(QColor(80, 80, 80), 2));    // 边框：深灰 2px
slot->setBrush(QColor(50, 50, 50));           // 填充：深灰
slot->setZValue(0.0);                         // 同棋盘格层级
```
